# 🚀 Smart CP Coach

A full-stack competitive programming tracker with AI-powered coaching, Codeforces sync, and performance insights.

---

## ✨ Features

- 📊 Dashboard with problem stats (solved, revision, unsolved)
- 🤖 AI Coach for guidance and suggestions
- 🔗 Codeforces integration (auto sync submissions)
- 🧠 Weak topic detection based on mistakes
- 📝 Manual problem tracking (notes, tags, status)
- ⚡ Smooth UI with AJAX + toast notifications

---

## 🛠️ Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite
- **APIs:** Codeforces API, OpenRouter (AI)
